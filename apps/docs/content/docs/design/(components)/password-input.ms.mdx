---

![password-input-anatomy.png](/design/password-input-anatomy.png)

1. Label Ruangan Kata Laluan.
2. Placeholder Asal: “Masukkan kata laluan”.
3. Ikon Tunjuk/Sorok:
   - Ikon Mata Tunjuk: Paparkan kata laluan.
   - Ikon Mata Sorok: Sorokkan kata laluan.

## Ruangan Pencetus Ikon Tunjuk/Sorok

![password-input-trigger-area.png](/design/password-input-trigger-area.png)

1. Tingkatkan ruangan pencetus untuk menyesuaikan ketinggian ruangan input secara seimbang, memberikan ruang yang mencukupi untuk pengguna mengklik ikon Tunjuk/Sorok.
2. Jika kata laluan disorok, paparkan tooltip dengan teks "Tunjuk kata laluan" apabila pengguna meletakkan tetikus di atas ruangan pencetus.
3. Jika kata laluan dipaparkan, paparkan tooltip dengan teks "Sorok kata laluan" apabila pengguna meletakkan tetikus di atas ruangan pencetus.

## Keadaan (State)

![password-input-state.png](/design/password-input-state.png)

1. **Asal**: Kata laluan disorok.
2. **Asal Berfokus**: Pengguna mengklik atau menggunakan papan kekunci untuk memasuki ruangan kata laluan, menunjukkan bahawa ia aktif dan sedia untuk menerima input.
3. **Terisi (Filled)**: Kata laluan dimasukkan, menandakan ruangan tersebut mengandungi input.
   3a. **Ikon Mata Tunjuk**: Klik untuk memaparkan kata laluan.
4. **Terisi (Kata Laluan Dipaparkan)**: Kata laluan ditunjukkan kepada pengguna.
   4a. **Ikon Mata Sorok**: Klik untuk menyembunyikan kata laluan.
5. **Ralat**: Paparkan garis besar berwarna merah (otl-danger-300) dan teks ralat apabila input kata laluan kosong.
6. **Ralat Berfokus**: Pengguna mengklik atau menggunakan papan kekunci untuk memasuki ruangan kata laluan yang dalam keadaan ralat, menunjukkan bahawa ia sedia untuk menerima input.

## Saiz

Setiap saiz direka untuk memenuhi keperluan borang yang berbeza, memastikan keterbacaan dan pengalaman pengguna yang optimum.
