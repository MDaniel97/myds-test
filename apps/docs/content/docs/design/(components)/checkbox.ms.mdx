Ia sering digunakan dalam borang, tetapan, dan penapis untuk menangkap input dan keutamaan pengguna.

## Saiz

Menentukan dimensi kotak semak untuk kesesuaian dalam pelbagai saiz borang.

![checkbox-size@1.5x.png](/design/checkbox-size1.5x.png)

## Keadaan (State)

Menunjukkan status semasa kotak semak, seperti dipilih atau tidak dipilih.

![checkbox-state@1.5x.png](/design/checkbox-state1.5x.png)

## Sifat (Property)

### Dipilih (Checked)

![checkbox-checked@1.5x.png](/design/checkbox-checked1.5x.png)

Sifat boolean yang menentukan sama ada kotak semak dipilih (true) atau tidak dipilih (false). Apabila dipilih, kotak semak akan memaparkan tanda semak.

1. **Ruangan Pencetus (Trigger Area)**: Tambah 5px padding di sekitar kotak semak untuk kemudahan klik.

### Indeterminate

![checkbox-intermediate@1.5x.png](/design/checkbox-intermediate1.5x.png)

Menunjukkan keadaan pilihan campuran atau sebahagian, sering digunakan dalam pilihan hierarki atau apabila status kotak semak menggambarkan gabungan item yang dipilih dan tidak dipilih. Biasanya ditunjukkan dengan tanda sengkang.

## Saiz label

![checkbox-label-size@1.5x.png](/design/checkbox-label-size1.5x.png)

Menentukan saiz kotak semak dan teks labelnya untuk kesesuaian dalam pelbagai saiz borang.

## Teks Petunjuk

![checkbox-hint-text@1.5x.png](/design/checkbox-hint-text1.5x.png)

Teks petunjuk di bawah label kotak semak menyediakan maklumat atau panduan tambahan kepada pengguna tentang pilihan kotak semak tersebut.
