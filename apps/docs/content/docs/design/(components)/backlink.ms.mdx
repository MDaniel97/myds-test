---
title: Pautan kembali
description: Pautan kembali digunakan untuk membantu pengguna kembali ke halaman sebelumnya dalam aliran pelbagai halaman. Pautan kembali boleh meningkatkan pautan dalaman, membantu carian enjin dan mengindeks laman web anda dengan lebih efektif. Ini dapat mengagihkan ekuiti pautan di seluruh halaman, sekaligus meningkatkan SEO secara keseluruhan.
---

## Keadaan (State)

![backlink-usage-example@1.5x.png](/design/backlink-usage-example1.5x.png)

1. Pautan kembali akan digariskan apabila tetikus berada di atasnya untuk menunjukkan interaktiviti.
2. Pautan kembali kontekstual hendaklah mengikuti format berikut: "\<- Kembali ke Halaman".
