Pill boleh mengandungi teks dan mungkin termasuk butang "x" untuk memudahkan penyingkiran.

## Saiz

![pills-size@1.5x.png](/design/pills-size1.5x.png)

Sifat saiz membantu memastikan pil diukur dengan tepat dan konsisten secara visual dengan elemen UI lain.

## Dilumpuh

![pills-disabled@1.5x.png](/design/pills-disabled1.5x.png)

Apabila pil dalam keadaan lumpuh, sembunyikan ikon "x" atau ikon penyingkiran untuk menunjukkan bahawa ia tidak interaktif.
