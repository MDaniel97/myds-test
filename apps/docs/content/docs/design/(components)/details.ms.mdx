## Keadaan (State)

![details-state@1.5x.png](/design/details-state1.5x.png)

1. **Ruangan Pencetus Klik** (Tajuk): Kawasan tajuk yang boleh diklik untuk membuka atau menutup butiran.
2. **Keadaan Hover**: Gariskan teks tajuk apabila tetikus berada di atasnya untuk menunjukkan interaktiviti.
3. **Keadaan Tertutup (Default)**: Menyembunyikan teks badan (Kebolehlihatan: 0%).
4. **Keadaan Terbuka**: Klik pada kawasan tajuk untuk membuka dan memaparkan teks badan dalam butiran.
