---
title: Pill
description: Represent tags or categories in a textfield. Pills can contain text and may include an "x" button to allow for easy removal.
---

## Size

![pills-size.png](/design/pills-size.png)

Size properties help ensure that the pills are appropriately proportioned and visually consistent with other UI elements.

## Disabled

![pills-disabled.png](/design/pills-disabled.png)

When a pill is disabled, hide the "x" or remove icon to indicate it is not interactive.
