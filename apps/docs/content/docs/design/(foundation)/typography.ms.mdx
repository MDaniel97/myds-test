## Tajuk

Keluarga fon [Poppins](https://fonts.google.com/specimen/Poppins) digunakan untuk tajuk seksyen halaman utama, tajuk halaman, dan elemen teks penting untuk mencipta hierarki visual yang jelas dan memperbaiki navigasi pengguna. Tidak terpakai untuk teks kaya.

![heading-typography.png](/design/heading-typography.png)

## Format Teks Kaya (RTF)

Keluarga fon [Inter](https://fonts.google.com/specimen/Inter?query=Inter) digunakan untuk penataan kandungan bentuk panjang seperti artikel.

**Nota:** Tag H1 dalam Format Teks Kaya (RTF) berbeza daripada tag Tajuk H1 standard. Penataan ini adalah khusus untuk pemformatan kandungan dalam bahagian artikel sahaja.

![rtf-typography.png](/design/rtf-typography.png)

### Saiz dan Berat Fon

Setiap saiz termasuk berat fon dalam Regular (400) dan Semibold (600) sahaja.

| Nama          | HTML Tag | Saiz Fon        | Ketinggian Garis | Jarak Senarai  | Jarak Perenggan |
| ------------- | -------- | --------------- | ---------------- | -------------- | --------------- |
| **Heading 1** | `<h1>`   | 30px (1.875rem) | 38px (2.375rem)  | 6px (0.375rem) | 12px (0.75rem)  |
| **Heading 2** | `<h2>`   | 24px (1.5rem)   | 32px (2rem)      | 6px (0.375rem) | 12px (0.75rem)  |
| **Heading 3** | `<h3>`   | 20px (1.25rem)  | 28px (1.75rem)   | 6px (0.375rem) | 12px (0.75rem)  |

| **Heading 5** | `<h5>`   | 16px (1rem)     | 24px (1.5rem)    | 6px (0.375rem) | 12px (0.75rem)  |
| **Heading 6** | `<h6>`   | 14px (0.875rem) | 20px (1.25rem)   | 6px (0.375rem) | 12px (0.75rem)  |
