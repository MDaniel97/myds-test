## Usage

```ts copy title="Import"
import { ThemeProvider } from "@govtechmy/myds-react/hooks";
import { ThemeSwitch } from "@govtechmy/myds-react/theme-switch";
```

```tsx copy title="Anatomy"
export default () => (
  <ThemeProvider>
    <ThemeSwitch />
  </ThemeProvider>
);
```

## Examples

### As Toggle

By default, `ThemeSwitch` will appear as a toggle button, switching between `light` and `dark` mode, just like below.

<Tabs items={["Preview", "Code"]} defaultValue="Preview">
  <Tab value="Preview">
    <div className="flex justify-center">
      <ThemeSwitch as="toggle" />
    </div>

  </Tab>
  <Tab value="Code">
```tsx
import { ThemeProvider } from "@govtechmy/myds-react/hooks";
import { ThemeSwitch } from "@govtechmy/myds-react/theme-switch";

<ThemeProvider>
  <ThemeSwitch as="toggle" />
</ThemeProvider>
```

</Tab>
</Tabs>
<Callout
  type="info"
  title="Important"
  className="bg-bg-primary-50 text-txt-primary border-transparent"
  icon={<InfoIcon className="size-5" />}
>
  Make sure your application is wrapped with `ThemeProvider` before using
  `ThemeSwitch` component.
</Callout>
