## Usage

```ts copy title="Import"
import { Pill } from "@govtechmy/myds-react/pill";
```

```tsx copy title="Anatomi"
export default () => <Pill>Trending</Pill>;
```

## Examples

### Size

Gunakan prop `size` untuk menukar saiz pil.

<Tabs items={["Preview", "Code"]} defaultValue="Preview">
  <Tab value="Preview" className="not-prose flex justify-center items-end gap-1">
      <Pill size="small">small</Pill>
      <Pill size="medium">medium</Pill>
      <Pill size="large">large</Pill>
  </Tab>
  <Tab value="Code">
    ```tsx 
    import { Pill } from "@govtechmy/myds-react/pill";

    <Pill size="small">small</Pill>;
    <Pill size="medium">medium</Pill>;
    <Pill size="large">large</Pill>;
    ```

  </Tab>
</Tabs>

### Dengan Butang "x" Mengekori

Hantarkan fungsi panggil balik ke prop `onDismiss` untuk menunjukkan butang "x" di belakang.

<Tabs items={["Preview", "Code"]} defaultValue="Preview">
  <Tab value="Preview" className="not-prose flex justify-center items-end gap-1">
      <PillWithTrailingXButton>Dengan butang "x" mengekori</PillWithTrailingXButton>
  </Tab>
  <Tab value="Code">
    ```tsx 
    import { Pill } from "@govtechmy/myds-react/pill";

    <Pill onDismiss={...}>Dengan butang "x" mengekori</Pill>
    ```

  </Tab>
</Tabs>

## Dinyahaktifkan

Pil boleh dinyahaktifkan dengan menetapkan prop `disabled` kepada `true`. Apabila dinyahaktifkan, pil tidak akan interaktif. Jika prop `onDismiss` ditetapkan, butang "x" akan disembunyikan.

<Tabs items={["Preview", "Code"]} defaultValue="Preview">
  <Tab value="Preview" className="not-prose flex justify-center items-end gap-1">
      <PillWithTrailingXButton>Enabled</PillWithTrailingXButton>
      <PillWithTrailingXButton disabled>Disabled</PillWithTrailingXButton>
  </Tab>
  <Tab value="Code">
    ```tsx 
    import { Pill } from "@govtechmy/myds-react/pill";

    <Pill onDismiss={...}>Enabled</Pill>
    <Pill disabled onDismiss={...}>Disabled</Pill>
    ```

  </Tab>
</Tabs>
