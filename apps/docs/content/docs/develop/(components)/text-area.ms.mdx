## Penggunaan

```tsx copy
import { TextArea } from "@govtechmy/myds-react/textarea";
```

```tsx copy
export default () => (
  <TextArea placeholder="Placeholder text displayed when the text area is empty." 
  size={'small'} 
  disabled={false}
  />
);
```

## Contoh
### Size
Mengawal varian saiz komponen. Menerima nilai `small`, `medium`, atau `large`.
<Tabs items={["Pratonton", "Kod"]} defaultValue="Pratonton">
  <Tab value="Pratonton">
  <div className="space-y-2">
  <div className="flex flex-col justify-center items-center h-fit gap-5">
      <TextArea size="small" placeholder="Enter your text for your small TextArea."/>
      <TextArea size="medium" placeholder="Enter your text for your medium TextArea."/>
      <TextArea size="large"placeholder="Enter your text for your large TextArea." />
      </div>

  </div>
  </Tab>
  <Tab value="Kod">
  ```tsx 
import { TextArea } from "@govtechmy/myds-react/textarea";

/** Size props*/

<TextArea size="small" placeholder="Enter your text for your small TextArea."/>
<TextArea size="medium" placeholder="Enter your text for your medium TextArea."/>
<TextArea size="large"placeholder="Enter your text for your large TextArea." />
```
  </Tab>
</Tabs>

### Disabled
Props in boleh digunakan untuk menyahaktifkan input pengguna dan menggunakan gaya visual untuk menunjukkan medan ini tidak boleh disunting.
<Tabs items={["Pratonton", "Kod"]} defaultValue="Pratonton">
  <Tab value="Pratonton">
  <div className="space-y-2">

      <TextArea size="small" placeholder="The TextArea has been disabled." disabled/>
      <TextArea size="medium" placeholder="The TextArea has been disabled." disabled/>
      <TextArea size="large"placeholder="The TextArea has been disabled." disabled/>
      </div>

  </div>
  </Tab>
  <Tab value="Kod">
  ```tsx

<TextArea size="small" placeholder="The TextArea has been disabled." disabled/>
<TextArea size="medium" placeholder="The TextArea has been disabled." disabled/>

```
  </Tab>
</Tabs>
