## Usage

```tsx copy
import { InputOTP, InputOTPSlot } from "@/components/myds"
```

```tsx copy
<InputOTP maxLength={4}>
  <InputOTPSlot index={0} />
  <InputOTPSlot index={1} />
  <InputOTPSlot index={2} />
  <InputOTPSlot index={3} />
</InputOTP>
```

## Contoh

Tetapkan prop `invalid` kepada true untuk menandakan input sebagai tidak sah.

    </InputOTP>
  </Tab>

  </Tab>

### Komponen Terkawal

Input boleh dikawal dengan menetapkan props `value` dan `onChange`. Dalam contoh di bawah, kami mengawal `InputOTP` dengan _state_ dan menukar nilai kepada huruf besar pada perubahan.

<Tabs items={["Preview", "Code"]} defaultValue="Preview">
  <Tab value="Preview" className="not-prose flex justify-center">
    <InputOTPControlledExample />
  </Tab>
  <Tab value="Code">
    ```tsx
    import { InputOTP, InputOTPSlot } from "@/components/myds"

    const [value, setValue] = useState("");

    <InputOTP
      maxLength={4}
      value={value}
      onChange={(value) => {
        setValue(value.toUpperCase());
      }}
    >
      <InputOTPSlot index={0} />
      <InputOTPSlot index={1} />
      <InputOTPSlot index={2} />
      <InputOTPSlot index={3} />
    </InputOTP>
    ```
  </Tab>
</Tabs>

### Menggunakan Prop `pattern`

Hantar regex kepada prop `pattern` untuk hanya menerima nilai corak tertentu. Sebagai contoh, kita boleh menggunakan regex untuk hanya menerima nombor:

<Tabs items={["Preview", "Code"]} defaultValue="Preview">
  <Tab value="Preview" className="not-prose flex justify-center">
    <InputOTP maxLength={4} pattern="^[0-9]{0,4}$">
      <InputOTPSlot index={0} />
      <InputOTPSlot index={1} />
      <InputOTPSlot index={2} />
      <InputOTPSlot index={3} />
    </InputOTP>
  </Tab>
  <Tab value="Code">
    ```tsx
    import { InputOTP, InputOTPSlot } from "@/components/myds"

    <InputOTP maxLength={4} pattern="^[0-9]{0,4}$">
      <InputOTPSlot index={0} />
      <InputOTPSlot index={1} />
      <InputOTPSlot index={2} />
      <InputOTPSlot index={3} />
    </InputOTP>
    ```
  </Tab>
</Tabs>
