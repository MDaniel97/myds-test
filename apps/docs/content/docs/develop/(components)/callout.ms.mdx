### Boleh Dihapuskan

Jika panggilan keluar boleh dihapuskan, anda boleh memaparkan **butang tutup** di sudut kanan atas panggilan keluar dengan mengisytiharkan prop `dismissible`.
Untuk menangkap peristiwa hapus, anda boleh menggunakan prop `onDismiss`.

Untuk menghapuskan callout melalui pendengar acara, balut elemen sasaran dengan `CalloutClose`. `CalloutClose` juga akan mencetuskan acara onDismiss, jika ditakrifkan.

<Tabs items={["Pratonton", "Kod"]} defaultValue="Pratonton">
  <Tab value="Pratonton">
    <div className="not-prose space-y-2">
      <DismissibleCalloutExample
        type="basic"
        title="Asas - Keadaan Hapus Tidak Terkawal"
        description="Anda boleh menangkap peristiwa hapus menggunakan prop onDismiss. Semak konsol untuk peristiwa hapus."
      />
      <DismissibleCalloutExample
        type="action"
        title="Tindakan - Hapus Callout melalui Prop Render CalloutAction"
        description="Anda boleh memanggil peristiwa hapus secara programatik melalui mana-mana pendengar peristiwa di dalam anak CalloutAction"
      />
    </div>
  </Tab>
  <Tab value="Kod">
    ```tsx
    // Asas - Keadaan Hapus Tidak Terkawal
    <Callout
      dismissible
      onDismiss={() => {
        console.log("MYDS: Peristiwa hapus ditangkap!");
      }}
    >
      <CalloutTitle>Tajuk</CalloutTitle>
      <CalloutContent>
        <span>
          Penerangan untuk menjelaskan kejayaan, amaran, maklumat atau peristiwa ralat
        </span>
      </CalloutContent>
      <CalloutAction>
        <CalloutClose>
        <Button variant="default-outline">Panggilan untuk Bertindak</Button>
        </CalloutClose>
      </CalloutAction>
    </Callout>

    // Tindakan - Hapus Callout melalui Prop Render CalloutAction
    <Callout
      dismissible
      onDismiss={() => {
        console.log("MYDS: Peristiwa hapus ditangkap!");
      }}
    >
    ```
