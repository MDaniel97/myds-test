## Cara Penggunaan

```tsx copy
import { Link } from "@govtechmy/myds-react/link";
```

```tsx copy title="Anatomi"
export default () => (
  <Link href="/" primary>
    {/* Destinasi pautan harus dinyatakan di sini. */}
    Sistem Reka Bentuk Rasmi Malaysia
  </Link>
);
```

## Contoh

### Prop asChild

Sekiranya anda perlu menggunakan komponen `Link` daripada pakej yang lain, anda boleh
menggunakan prop `asChild` seperti berikut (Next.js sebagai contoh):

<Tabs items={["Pratonton", "Kod"]} defaultValue="Preview">
  <Tab value="Pratonton">
    <Link asChild>
      <NextLink href="#using-aschild-for-custom-links">
        Custom Link
      </NextLink>
    </Link>
  </Tab>
  <Tab value="Kod">
    ```tsx 
    import { Link } from "@govtechmy/myds-react/link";
    import NextLink from "next/link";

    <Link asChild>
      <NextLink href="#">Custom Link</NextLink>
    </Link>
    ```

  </Tab>
</Tabs>

Anda juga boleh membina komponen sendiri dan import mengikut keperluan:

```tsx
import { Link as LinkPrimitive } from "@govtechmy/myds-react/link";
import NextLink from "next/link";

export const Link = (props) => (
  <LinkPrimitive asChild>
    <NextLink {...props} />
  </LinkPrimitive>
);
```

### Prop newTab

Prop ini menetapkan sama ada pautan dibuka dalam tab baharu dengan menggunakan `target="_blank"`.

<Tabs items={["Pratonton", "Kod"]} defaultValue="Preview">
  <Tab value="Pratonton">
    <Link href="https://digital.gov.my" newTab>Ministry of Digital</Link>
  </Tab>
  <Tab value="Kod">
    ```tsx 
    import { Link } from "@govtechmy/myds-react/link";

    <Link href="https://digital.gov.my" newTab>Ministry of Digital</Link>
    ```

  </Tab>
</Tabs>

<Callout
  type="info"
  title="Note"
  className="bg-bg-primary-50 text-txt-primary border-transparent"
  icon={<InfoIcon className="size-5" />}
>
  Semua versi semasa pelayar web terkini secara automatik menggunakan
  `rel="noopener"` bagi semua pautan mempunyai `target="_blank"`
</Callout>

### Primary

Prop ini digunakan untuk menetapkan warna teks pautan.

<Tabs items={["Pratonton", "Kod"]} defaultValue="Preview">
  <Tab value="Pratonton">
    <div className="space-y-2">
      <div className="flex justify-center items-center gap-2">
        <Link href="#primary" primary>Primary</Link>
        <Link href="#primary">Inherit from parent (default)</Link>
      </div>
    </div>
  </Tab>
  <Tab value="Kod">
    ```tsx 
    import { Link } from "@govtechmy/myds-react/link";

    <Link href="#" primary>Primary</Link>
    <Link href="#">Inherit from parent</Link>
    ```

  </Tab>
</Tabs>

### Underline

Prop ini digunakan untuk menetapkan waktu pautan digaris bawah.

<Tabs items={["Pratonton", "Kod"]} defaultValue="Preview">
  <Tab value="Pratonton">
    <div className="space-y-2">
      <div className="flex justify-center items-center h-fit gap-2 not-prose">
        <Link href="#underline" underline="always" >Always</Link>
        <Link href="#underline" underline="hover" >Hover</Link>
        <Link href="#underline" underline="none" >None</Link>
      </div>
    </div>
  </Tab>
  <Tab value="Kod">
    ```tsx 
    import { Link } from "@govtechmy/myds-react/link";

    <Link href="#underline" underline="always">Always</Link>
    <Link href="#underline" underline="hover">Hover</Link>
    <Link href="#underline" underline="none">None</Link>
    ```

  </Tab>
</Tabs>
