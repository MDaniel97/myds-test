## Penggunaan

```ts copy title="Import"
import {
  Tooltip,
  TooltipTrigger,
  TooltipContent,
} from "@govtechmy/myds-react/tooltip";
```

```tsx copy title="Anatomy"
<Tooltip>
  <TooltipTrigger />
  <TooltipContent />
</Tooltip>
```

## Contoh-contoh

### Keadaan Tidak Terkawal vs Terkawal

Untuk menggunakan komponen ini dalam keadaan tidak terkawal, tetapkan prop `defaultOpen` kepada keadaan awal input.

Untuk menggunakan komponen dalam keadaan terkawal, gunakan prop `open` dan `onOpenChange` untuk mengendalikan keadaan secara programatik.

<Tabs items={["Pratonton", "Kod"]} defaultValue="Pratonton">
  <Tab value="Pratonton" className="not-prose flex justify-center">
    <Tooltip open={true}>
      <TooltipTrigger>
          <QuestionCircleIcon />
      </TooltipTrigger>
      <TooltipContent>Ini merupakan penjelasan ringkas.</TooltipContent>
  </Tooltip>
  </Tab>
  <Tab value="Kod">
    ```tsx 
    import { Tooltip, TooltipTrigger, TooltipContent } from "@govtechmy/myds-react/tooltip";

    <Tooltip open={true}>
      <TooltipTrigger>
          <QuestionCircleIcon />
      </TooltipTrigger>
      <TooltipContent>Ini merupakan penjelasan ringkas.</TooltipContent>

  </Tooltip>
    ```
  </Tab>
</Tabs>

## Penempatan Tooltip

Props `side` boleh digunakan untuk melaraskan kedudukan `TooltipContent`. Nilai-nilai yang diterima ialah: `left`, `right`, `top` dan `bottom`.

<Tabs items={["Pratonton", "Kod"]} defaultValue="Pratonton">
  <Tab value="Pratonton" className="not-prose flex flex-col justify-center items-center gap-10">
    <Tooltip>
      <TooltipTrigger>
          <QuestionCircleIcon />
      </TooltipTrigger>
      <TooltipContent side={"left"}>Penjelasan terletak di sebelah kiri.</TooltipContent>
    </Tooltip>
    <Tooltip>
        <TooltipTrigger>
            <QuestionCircleIcon />
        </TooltipTrigger>
        <TooltipContent side={"right"}>Penjelasan terletak di sebelah kanan.</TooltipContent>
    </Tooltip>
    <Tooltip>
        <TooltipTrigger>
            <QuestionCircleIcon />
        </TooltipTrigger>
        <TooltipContent side={"top"}>Penjelasan terletak di sebelah atas.</TooltipContent>
    </Tooltip>
    <Tooltip>
        <TooltipTrigger>
            <QuestionCircleIcon />
        </TooltipTrigger>
        <TooltipContent side={"bottom"}>Penjelasan terletak di sebelah bawah.</TooltipContent>
    </Tooltip>
  </Tab>

## Props

Komponen `Tooltip` dibina di atas `TooltipPrimitive.Provider` and `TooltipPrimitive.Root`. Sila rujuk senarai API lengkap dalam [documentasi ini](https://www.radix-ui.com/primitives/docs/components/tooltip#provider).

### TooltipContent

Komponen `TooltipContent` ini dibina di atas `TooltipPrimitive.Portal` dan `TooltipPrimitive.Content`. Sila rujuk senarai penuh API di [dokumentasi ini](https://www.radix-ui.com/primitives/docs/components/tooltip#portal).

### TooltipTrigger

`TooltipTrigger` membolehkan pengguna untuk berinteraksi atas tujuan menunjukkan "tooltip". Komponenen ini dibina menggunakan `TooltipPrimitive.Trigger`. Sila rujuk senarai penuh API di [dokumentasi ini](https://www.radix-ui.com/primitives/docs/components/tooltip#content).
