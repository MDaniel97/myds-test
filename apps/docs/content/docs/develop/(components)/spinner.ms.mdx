## Penggunaan

```ts copy title="Import"
import { Spinner } from "@govtechmy/myds-react/spinner";
```

```tsx copy title="Anatomi"
export default () => <Spinner />;
```

## Contoh

### Varian

Gunakan prop `color` untuk menukar gaya pemuat!

<Tabs items={["Pratonton", "Kod"]} defaultValue="Pratonton">
  <Tab value="Pratonton">
    <div className="space-y-2">
        <div className="flex items-center justify-center">
            <Button variant="default-outline">Memuatkan Kelabu <Spinner/></Button>
        </div>
        <div className="flex items-center justify-center">
            <Button>Memuatkan Putih<Spinner color={"white"}/></Button>
        </div>
        <div className="flex items-center justify-center">
            <Button variant="default-outline">Memuatkan Default <Spinner/></Button>
        </div>
  </div>
  </Tab>
  <Tab value="Kod">
  ```tsx 
import { Spinner } from "@govtechmy/myds-react/spinner";

/** Warna: Kelabu **/

<Button variant="default-outline">
  Memuatkan Kelabu
  <Spinner color={"gray"} />
</Button>

/** Warna: Putih **/

<Button>
  Memuatkan Putih
  <Spinner color={"white"} />
</Button>

/** Varian: Default **/

<Button variant="default-outline">
  Memuatkan Default
  <Spinner />
</Button>

````
  </Tab>
</Tabs>

### Paparkan

Gunakan prop `show` untuk memaparkan atau menyembunyikan spinner.

<Tabs items={["Pratonton", "Kod"]} defaultValue="Pratonton">
  <Tab value="Pratonton">
  <div className="space-y-2">
  <div className="flex justify-center items-center h-fit gap-2">

      <Button variant="default-outline">
        <span>Spinner Show</span><Spinner show={true}/>
      </Button>

      <Button variant="default-outline">
        <span>Spinner Hide</span><Spinner show={false}/>
      </Button>
      </div>

  </div>
  </Tab>
  <Tab value="Kod">
  ```tsx
import { Button } from "@govtechmy/myds-react/button";

/** Papar or Sembunyi **/

<Button variant="default-outline">
  <span>Spinner Dipaparkan</span>
  <Spinner show={true} />
</Button>

<Button variant="default-outline">
  <span>Spinner Disembunyikan</span>
  <Spinner show={false} />
</Button>
```

  </Tab>
</Tabs>
