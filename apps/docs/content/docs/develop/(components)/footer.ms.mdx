## Penggunaan

```ts copy title="Import"
import {
  Footer,
  SiteInfo,
  FooterSection,
  SiteLinkGroup,
  SiteLink,
  FooterLogo,
} from "@govtechmy/myds-react/footer";
```

```tsx copy title="Anatomy"
export default () => (
  <Footer>
    <FooterSection>
      <SiteInfo>
        <FooterLogo />
      </SiteInfo>
      <SiteLinkGroup>
        <SiteLink />
      </SiteLinkGroup>
    </FooterSection>
  </Footer>
);
```

### 1.2 FooterContent

Komponen `FooterContent` direka untuk mengumpulkan pautan yang berkaitan dalam susun atur yang tersusun, biasanya diatur dalam lajur. Setiap lajur diwakili oleh `FooterContentColumn`, yang merangkumi tajuk dan pautan yang berkaitan.

**Contoh**:


<Tabs items={["Preview", "Code"]} defaultValue="Preview" className="not-prose">

## 2. FooterBottomSection

`FooterBottomSection` adalah bahagian bawah footer. Ia mengandungi yang berikut:

1.1 **FooterCopyright**  
1.2 **FooterTimestamp**

### 2.1 FooterCopyright

Kawasan `FooterCopyright` memaparkan maklumat hak cipta dan pautan berkaitan tambahan. Ia termasuk komponen berikut:

#### 2.1.1 FooterCopyrightDate

Memaparkan kenyataan hak cipta dengan tahun.

**Contoh**:

<Tabs items={["Preview", "Code"]} defaultValue="Preview">
  <Tab value="Preview">
  <div className="flex justify-center items-center h-fit gap-1">
  <FooterCopyrightDate>Hak Cipta Terpelihara</FooterCopyrightDate>
  </div>

  </Tab>
  <Tab value="Code">
```tsx
<FooterCopyrightDate>Hak Cipta Terpelihara</FooterCopyrightDate>
```
  </Tab>
</Tabs>
