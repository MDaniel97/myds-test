## Usage

```ts copy title="Import"
import {
  SummaryList,
  SummaryListAction,
  SummaryListBody,
  SummaryListHeader,
  SummaryListTerm,
  SummaryListDetail,
  SummaryListRow,
} from "@govtechmy/myds-react/summary-list";
```

```tsx copy title="Anatomy"
export default () => (
  <SummaryList>
    <SummaryListHeader />
    <SummaryListBody>
      <SummaryListRow>
        <SummaryListTerm />
        <SummaryListDetail />
        <SummaryListAction />
      </SummaryListRow>
    </SummaryListBody>
  </SummaryList>
);
```

---

### Main Wrapper: `SummaryList`

The `SummaryList` is the primary container for presenting a summary of items in a structured and visually consistent format. It serves as the parent component, wrapping all subcomponents.

1. `SummaryListHeader`
2. `SummaryListBody`

---

## 1. SummaryListHeader

The `SummaryListHeader` is used to display the title of the summary list.

**Example**:

<Tabs items={["Preview", "Code"]} defaultValue="Preview">
  <Tab value="Preview">
    <SummaryListHeader>Government Subsidy Application</SummaryListHeader>
  </Tab>

  <Tab value="Code">

  </Tab>
</Tabs>

## 2. SummaryListBody

The SummaryListBody is a wrapper for the tabular data within the SummaryList. Main wrapper for multiple or single SummaryListRow.

**Example**:
