## Penggunaan

```ts copy title="Import"
import {
  Select,
  SelectGroup,
  SelectGroupTitle,
  SelectValue,
  SelectTrigger,
  SelectLabel,
  SelectContent,
  SelectHeader,
  SelectFooter,
  SelectItem,
  SelectSeparator,
} from "@govtechmy/myds-react/select";
```

```tsx copy title="Anatomi"
export default () => (
  <Select>
    <SelectTrigger>
      <SelectValue />
    </SelectTrigger>
    <SelectContent>
      <SelectHeader />
      <SelectGroup>
        <SelectGroupTitle />
        <SelectItem />
      </SelectGroup>
      <SelectSeparator />
      <SelectFooter />
    </SelectContent>
  </Select>
);
```

### Dinonaktifkan

Terdapat banyak cara untuk berinteraksi dengan keadaan `disabled` Pilih. `Select` boleh dinonaktifkan sepenuhnya dengan menetapkan prop `disabled` kepada `true`.

Untuk kawalan yang lebih halus pada pilihan mana yang hendak dinonaktifkan, anda boleh menetapkan `disabled` pada `SelectItem` individu!

### Paparan Nilai Tersuai

Anda boleh menyesuaikan paparan nilai dengan menggunakan komponen `SelectValue`. Ia membolehkan anda menyesuaikan paparan nilai pilihan.

<Tabs items={["Pratonton", "Kod"]} defaultValue="Pratonton">
  <Tab value="Pratonton">
    <div className="not-prose flex justify-center gap-3">
        <CustomValueSelect />
    </div>

  </Tab>
  <Tab value="Kod">
    ```tsx
    <Select multiple size="small" variant="outline">
      <SelectTrigger>
        <SelectValue label="Buah">
          {(value) => (
            <div className="flex gap-1">
              {Array.isArray(value)
                ? value.map((v) => <Pill>{v}</Pill>)
                : value}
            </div>
          )}
        </SelectValue>
      </SelectTrigger>
      <SelectContent>
        <SelectItem value="apple">Epal</SelectItem>
        <SelectItem value="banana">Pisang</SelectItem>
        <SelectItem value="blueberry">Blueberry</SelectItem>
        <SelectItem value="grapes">Anggur</SelectItem>
        <SelectItem value="pineapple">Nanas</SelectItem>
      </SelectContent>
    </Select>
    ```
  </Tab>
