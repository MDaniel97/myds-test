## Usage

```ts copy title="Import"
import {
  CookieBanner,
  CookieBannerClose,
  CookieBannerCustomiser,
  CookieBannerDescription,
  CookieBannerFooter,
  CookieBannerHeader,
  CookieBannerPreferences,
  CookieBannerTitle,
} from "@govtechmy/myds-react/cookie-banner";
```

```tsx copy title="Anatomi"
export default () => (
  <CookieBanner>
    <CookieBannerHeader>
      <CookieBannerTitle />
      <CookieBannerClose />
    </CookieBannerHeader>
    <CookieBannerDescription />
    <CookieBannerPreferences />
    <CookieBannerFooter>
      <CookieBannerCustomiser />
    </CookieBannerFooter>
  </CookieBanner>
);
```

## Props

### CookieBanner

Menempatkan semua komponen konstituen untuk sepanduk kuki. Komponen ini adalah komponen terkawal. Ia dibina menggunakan `Dialog` dan `DialogContent`.

<TypeTable
  type={{

      typeDescription: "true | false",
      default: "false",
    },
  }}
/>

### CookieBannerHeader

Mengandungi komponen `CookieBannerTitle` dan `CookieBannerClose`. Ia dibina menggunakan `DialogHeader`.

### CookieBannerTitle

Digunakan untuk mengubah tajuk sepanduk kuki. Ia dibina menggunakan `DialogTitle`.
