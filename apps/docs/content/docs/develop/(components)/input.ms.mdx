## Penggunaan

```ts copy title="Import"
import { Input } from "@govtechmy/myds-react/input";
```

```tsx copy title="Anatomi"
<Input />
```

## Contoh

### Saiz

Gunakan prop `size` untuk menukar saiz input. Saiz lalai adalah `small`.

<Tabs items={["Pratonton", "Kod"]} defaultValue="Pratonton">
  <Tab value="Pratonton" className="not-prose block space-y-1">
  
    <Input size="small" placeholder="Kecil" />
    <Input size="medium" placeholder="Sederhana" />
    <Input size="large" placeholder="Besar" />
  </Tab>
  <Tab value="Kod">
    ```tsx
    <Input size="small" placeholder="Kecil" />
    <Input size="medium" placeholder="Sederhana" />
    <Input size="large" placeholder="Besar" />
    ```

  </Tab>
</Tabs>

### Dinonaktifkan

Nonaktifkan input dengan menetapkan prop `disabled` kepada true.

<Tabs items={["Pratonton", "Kod"]} defaultValue="Pratonton">
  <Tab value="Pratonton" className="not-prose flex justify-center">
    <Input disabled placeholder="Dinonaktifkan" />
  </Tab>
  <Tab value="Kod">
    ```tsx
    <Input disabled placeholder="Dinonaktifkan" />
    ```

  </Tab>
</Tabs>

### Keadaan Tidak Terkawal vs Terkawal

Untuk menggunakan input dalam keadaan tidak terkawal, tetapkan prop `defaultValue` kepada keadaan awal input.

Untuk menggunakan input dalam keadaan terkawal, gunakan prop `value` dan `onChange` untuk mengendalikan keadaan secara programatik.

<Tabs items={["Pratonton", "Kod"]} defaultValue="Pratonton">
  <Tab value="Pratonton" className="not-prose block space-y-2">
    <Input defaultValue="Tidak Terkawal" />
    <Input value="Terkawal" />
  </Tab>
  <Tab value="Kod">
    ```tsx
    <Input defaultValue="Tidak Terkawal" />
    <Input value="Terkawal" />
    ```

  </Tab>
</Tabs>

### Ikon Input

Anda boleh menambah ikon pada medan input dengan menggunakan komponen `InputIcon`. Ikon boleh diletakkan di sebelah `kiri` atau `kanan` medan input. Kedua-dua ikon kiri dan kanan boleh digunakan bersama.

<Tabs items={["Pratonton", "Kod"]} defaultValue="Pratonton">
  <Tab value="Pratonton" className="not-prose flex flex-col justify-center gap-2">
    <Input placeholder="Input + Ikon Kiri">
      <InputIcon position="left">
        <ArrowBackIcon />
      </InputIcon>
    </Input>
    <Input placeholder="Input + Ikon Kanan">
      <InputIcon position="right">
        <ArrowForwardIcon />
      </InputIcon>
    </Input>
    <Input placeholder="Input + Ikon Kiri & Kanan">
      <InputIcon position="left">
        <ArrowBackIcon />
      </InputIcon>
      <InputIcon position="right">
        <ArrowForwardIcon />
      </InputIcon>
    </Input>
  </Tab>
  <Tab value="Kod">
```tsx
// Ikon Kiri

<Input placeholder="Input + Ikon Kiri">
  <InputIcon position="left">
    <ArrowBackIcon />
  </InputIcon>
</Input>

// Ikon Kanan

<Input placeholder="Input + Ikon Kanan">
  <InputIcon position="right">
    <ArrowForwardIcon />
  </InputIcon>
</Input>

// Ikon Kiri & Kanan

<Input placeholder="Input + Ikon Kiri & Kanan">
  <InputIcon position="left">
    <ArrowBackIcon />
  </InputIcon>
  <InputIcon position="right">
    <ArrowForwardIcon />
  </InputIcon>
</Input>
```
  </Tab>
</Tabs>

### Tambahan Input

Anda boleh menambah tambahan pada medan input dengan menggunakan komponen `InputAddon`. Tambahan boleh ditambah atau disisipkan pada medan input. Ini berguna untuk melampirkan **butang hantar**, **simbol mata wang** atau sebarang kawalan lain pada medan input.

<Tabs items={["Pratonton", "Kod"]} defaultValue="Pratonton">
  <Tab value="Pratonton" className="not-prose flex flex-col gap-2">
    <Input
      placeholder="Taip sesuatu di sini"
      prepend={<InputAddon>RM</InputAddon>}
    />
    <Input
      placeholder="Taip sesuatu di sini"
      append={
        <InputAddon>
          <button>Hantar</button>
        </InputAddon>
      }
    />
  </Tab>
  <Tab value="Kod">
    ```tsx
    // Tambah
    <Input
      placeholder="Taip sesuatu di sini"
      prepend={<InputAddon>RM</InputAddon>}
    />

    // Sisip
    <Input
      placeholder="Taip sesuatu di sini"
      append={
        <InputAddon>
          <button>Hantar</button>
        </InputAddon>
      }
    />
    ```

  </Tab>
</Tabs>
