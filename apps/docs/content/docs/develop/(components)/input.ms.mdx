## Penggunaan

```tsx copy
import { Input } from "@govtechmy/myds-react/input";
```

```tsx copy
<Input />
```

## Contoh

### Saiz

Gunakan prop `size` untuk menukar saiz input. Saiz lalai adalah `small`.

<Tabs items={["Pratonton", "Kod"]} defaultValue="Pratonton">
  <Tab value="Pratonton" className="not-prose block space-y-1">
  
    <Input size="small" placeholder="Kecil" />
    <Input size="medium" placeholder="Sederhana" />
    <Input size="large" placeholder="Besar" />
  </Tab>
  <Tab value="Kod">
    ```tsx
    <Input size="small" placeholder="Kecil" />
    <Input size="medium" placeholder="Sederhana" />
    <Input size="large" placeholder="Besar" />
    ```

  </Tab>
</Tabs>

### Dinonaktifkan

Nonaktifkan input dengan menetapkan prop `disabled` kepada true.

<Tabs items={["Pratonton", "Kod"]} defaultValue="Pratonton">
  <Tab value="Pratonton" className="not-prose flex justify-center">
    <Input disabled placeholder="Dinonaktifkan" />
  </Tab>
  <Tab value="Kod">
    ```tsx
    <Input disabled placeholder="Dinonaktifkan" />
    ```

  </Tab>
</Tabs>

### Keadaan Tidak Terkawal vs Terkawal

Untuk menggunakan input dalam keadaan tidak terkawal, tetapkan prop `defaultValue` kepada keadaan awal input.

Untuk menggunakan input dalam keadaan terkawal, gunakan prop `value` dan `onChange` untuk mengendalikan keadaan secara programatik.

<Tabs items={["Pratonton", "Kod"]} defaultValue="Pratonton">
  <Tab value="Pratonton" className="not-prose block space-y-2">
    <Input defaultValue="Tidak Terkawal" />
    <Input value="Terkawal" />
  </Tab>
  <Tab value="Kod">
    ```tsx
    <Input defaultValue="Tidak Terkawal" />
    <Input value="Terkawal" />
    ```

  </Tab>
</Tabs>
