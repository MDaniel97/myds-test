## Penggunaan

```tsx copy
import {
  Breadcrumb,
  BreadcrumbItem,
  BreadcrumbLink,
  BreadcrumbSeparator,
  BreadcrumbPage,
} from "@govtechmy/myds-react/breadcrumb";
```

```tsx copy
export default () => (
  <Breadcrumb>
    <BreadcrumbItem>
      <BreadcrumbLink />
      <BreadcrumbPage />
    </BreadcrumbItem>
    <BreadcrumbSeparator />
  </Breadcrumb>
);
```

## Contoh

### Varian

Gunakan prop `variant` untuk menukar gaya breadcrumb!

<Tabs items={["Pratonton", "Kod"]} defaultValue="Pratonton">

   <Breadcrumb variant="default">
    <BreadcrumbItem>
      <BreadcrumbLink href="#home">Laman Utama</BreadcrumbLink>
    </BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem>
      <BreadcrumbLink href="#collection">Koleksi</BreadcrumbLink>

    <BreadcrumbSeparator />
    <BreadcrumbItem>
      <BreadcrumbPage>Laman Anda</BreadcrumbPage>

### Pautan vs Halaman

`BreadcrumbLink` digunakan untuk halaman induk, manakala `BreadcrumbPage` digunakan untuk halaman semasa. Halaman semasa biasanya tidak boleh diklik.

<Tabs items={["Pratonton", "Kod"]} defaultValue="Pratonton">
  <Tab value="Pratonton">
    <div className="not-prose space-y-2">
      <Breadcrumb variant="default">
        <BreadcrumbItem>
          <BreadcrumbLink href="#home">BreadcrumbLink 1</BreadcrumbLink>
        </BreadcrumbItem>
        <BreadcrumbSeparator />
        <BreadcrumbItem>
          <BreadcrumbLink href="#collection">BreadcrumbLink 2</BreadcrumbLink>
        </BreadcrumbItem>
        <BreadcrumbSeparator />
        <BreadcrumbItem>
          <BreadcrumbPage>BreadcrumbPage</BreadcrumbPage>
        </BreadcrumbItem>
      </Breadcrumb>
    </div>
  </Tab>
  <Tab value="Kod">
    ```tsx
    <Breadcrumb variant="default">
      <BreadcrumbItem>
        <BreadcrumbLink href="#home">Laman Utama</BreadcrumbLink>
      </BreadcrumbItem>
      <BreadcrumbSeparator />
      <BreadcrumbItem>
    ```

### Serpihan Ellipsis

Serpihan yang terlalu panjang akan dipendekkan dengan elipsis. Had lebar maksimum ialah `200px`. Pengguna boleh melayang di atas serpihan untuk melihat teks penuh.

<Tabs items={["Pratonton", "Kod"]} defaultValue="Pratonton">
  <Tab value="Pratonton">
    <div className="not-prose space-y-2">
      <Breadcrumb variant="default">
        <BreadcrumbItem>
          <BreadcrumbLink href="#home">Laman Utama</BreadcrumbLink>
        </BreadcrumbItem>
        <BreadcrumbSeparator />
        <BreadcrumbItem>
          <BreadcrumbLink href="#collection">
            Ini adalah nama koleksi yang sangat panjang untuk mencetuskan
            elipsis
          </BreadcrumbLink>
        </BreadcrumbItem>
        <BreadcrumbSeparator />
        <BreadcrumbItem>
          <BreadcrumbPage>Laman Anda</BreadcrumbPage>
        </BreadcrumbItem>
      </Breadcrumb>
    </div>
  </Tab>
  <Tab value="Kod">
