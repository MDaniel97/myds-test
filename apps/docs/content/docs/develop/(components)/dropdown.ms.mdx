## Penggunaan

```ts copy title="Import"
import {
  Dropdown,
  DropdownContent,
  DropdownItem,
  DropdownItemIcon,
  DropdownTrigger,
} from "@govtechmy/myds-react/dropdown";
```

```tsx copy title="Anatomi"
export default () => (
  <Dropdown>
    <DropdownTrigger />
    <DropdownContent>
      <DropdownItem>
        <DropdownItemIcon />
      </DropdownItem>
    </DropdownContent>
  </Dropdown>
);
```

### Keadaan Tidak Terkawal vs Terkawal

Untuk menggunakan menu dropdown dalam keadaan tidak terkawal, tetapkan prop `defaultOpen` kepada keadaan awal menu dropdown.

Untuk menggunakan menu dropdown dalam keadaan terkawal, gunakan prop `open` dan `onOpenChange` untuk mengendalikan keadaan secara programatik.

<Tabs items={["Pratonton", "Kod"]} defaultValue="Pratonton">
  <Tab value="Pratonton">
    <div className="not-prose flex flex-col gap-3 items-center">
        <div>
            Tidak Terkawal
             <Dropdown defaultOpen={false}>
                <DropdownTrigger asChild>
                    <Button variant="primary-fill">Menu Dropdown</Button>
                </DropdownTrigger>
                <DropdownContent>
                    <DropdownItem>Profil</DropdownItem>
                    <DropdownItem>Pengebilan</DropdownItem>
                    <DropdownItem>Pasukan</DropdownItem>
                    <DropdownItem>Langganan</DropdownItem>
                </DropdownContent>
            </Dropdown>
        </div>
        <div>
            Terkawal
            <Dropdown open={false}>
                <DropdownTrigger asChild>
                    <Button variant="primary-fill">Menu Dropdown</Button>
                </DropdownTrigger>
                <DropdownContent>
                    <DropdownItem>Profil</DropdownItem>
                    <DropdownItem>Pengebilan</DropdownItem>
                    <DropdownItem>Pasukan</DropdownItem>
                    <DropdownItem>Langganan</DropdownItem>
                </DropdownContent>
            </Dropdown>
        </div>
    </div>

  </Tab>
  <Tab value="Kod">
