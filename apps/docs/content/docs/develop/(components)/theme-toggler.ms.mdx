## Penggunaan

```tsx copy
import {
  ThemeProvider,
  ThemeToggler,
} from "@govtechmy/myds-react/theme-toggler";
```

```tsx copy
export default () => (
  <ThemeProvider>
    <ThemeToggler/>
  </ThemeProvider>
);
```

## Contoh

### Penggunaan Asas  

Wrap aplikasi anda dengan `ThemeProvider` dan gunakan `ThemeToggler` untuk bertukar antara tema.

<Tabs items={["Preview", "Code"]} defaultValue="Preview">
  <Tab value="Preview">
    <div className="flex justify-center">
        <SampleThemeToggler/>
    </div>
  </Tab>
  <Tab value="Code">
  ```tsx
    import { ThemeProvider, ThemeToggler } from "@govtechmy/myds-react/theme-toggler";
    import { MoonIcon, SunIcon } from "@govtechmy/myds-react/icon";

    <ThemeProvider>
        <ThemeToggler/>
    </ThemeProvider>
  ```
  </Tab>
</Tabs>

### Tema Tersuai  

Hantarkan satu array tema ke `ThemeToggler` untuk menyesuaikan tema yang tersedia.

<Tabs items={["Preview", "Code"]} defaultValue="Preview">
  <Tab value="Preview">
    <div className="flex justify-center">
     <CustomThemeToggler/>
    </div>
  </Tab>
  <Tab value="Code">
  ```tsx
 import { MoonIcon, SunIcon } from "@govtechmy/myds-react/icon";
 import {
  ThemeProvider,
  ThemeToggler,
 } from "@govtechmy/myds-react/theme-toggler";

    <ThemeProvider>
      <ThemeToggler
        themes={[
          { theme: "light", icon: <SunIcon /> },
          { theme: "dark", icon: <MoonIcon /> },
          { theme: "sephia", icon: <PlaceholderIcon /> },
          { theme: "retro", icon: <PlaceholderIcon /> },
        ]}
      />
    </ThemeProvider>
  ```
  </Tab>
</Tabs>
