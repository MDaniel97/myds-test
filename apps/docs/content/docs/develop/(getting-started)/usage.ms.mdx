### Komposisi<a name="composition"></a>

Komponen MYDS dibina berteraskan konsep komposisi. Komposisi menawarkan penyesuaian pada tahap terendah, memberikan kebebasan tertinggi kepada pembangun untuk menambah sentuhan tersendiri kepada MYDS.
Pada masa yang sama, teknik ini juga boleh mencapai yang bertentangan penyesuaian maksima, dengan 'mengganti' props untuk menerapkan kelas atau pendengar acara tersendiri.

#### A. Gantikan komponen induk

<Preview>
  <Button variant="primary-ghost" size="small" asChild>
    <Link href="#composition" className="no-underline">Kembali ke tajuk</Link>
  </Button>

</Preview>

```tsx title="Anchor dalam bentuk Button"
import { Button } from "@myds/react/button";
import { Link } from "@myds/react/link";

<Button variant="primary-ghost" size="small" asChild>
  <Link href="#composition">
    <span className="text-txt-primary">Kembali ke tajuk</span>
  </Link>
</Button>;
```

#### B. Ganti props

<Preview>
  <PreviewButton
    variant="primary-fill"
    size="small"
    pantun="Berjalan-jalan ke kota tua, Membeli kuih di pasar pagi. Membuka jalan menuju dunia, 'Button' ditekan dengan jari. -- Copilot"
  >
    <span className="text-txt-white">Jana Pantun</span>
    <ButtonIcon>
      <ArrowForward />
    </ButtonIcon>
  </PreviewButton>
</Preview>

```tsx title="Button dengan Icon Props yang Diganti"
import { Button, ButtonIcon } from "@myds/react/button";
import { ArrowForwardIcon } from "@myds/react/icon";

<Button variant="primary-fill" size="small" onClick={...}>
  Jana Pantun
  <ButtonIcon>
    <ArrowForward />
  </ButtonIcon>
</Button>;
```

Ini mungkin konsep baru bagi sesetengah pembangun, namun, ia sangat berbaloi untuk belajar & mahir.
Komposisi adalah teknik yang penting dan menawarkan nilai yang besar apabila membina komponen.
