### Pemasangan secara Manual

MYDS boleh ditetapkan dengan pelbagai cara, bergantung kepada keperluan projek anda. Namun, cara yang paling mudah mengambil hanya 3-langkah!

<Steps>
  <Step>
    ### Pemasangan pakej
   
     ```package-install
    npm i @govtechmy/myds-react @govtechmy/myds-style
    ```
  </Step>

  <Step>

    ### Tetapkan stylesheet CSS<a name="full-css"></a>

    Anda boleh 'import' stylesheet MYDS secara keseluruhan mengikut salah satu cara di bawah.

    __1. Import melalui JS (Bundler)__

    ```ts
    // Letakkan dalam fail permulaan JS/TS (eg. index.{ts,js})
    import "@govtechmy/myds-style/full.css";
    ```

    __2. Import melalui CSS__

    ```css
    /* Letakkan dalam fail permulaan CSS  */
    @import "@govtechmy/myds-style/full.css";
    ```

    Untuk tetapan yang lebih lanjut, sila baca [Konfigurasi Style](./style-configuration)

  </Step>

<Step>
    ### Mula menggunakan komponen!

    Cubalah `<Button />` untuk memulakan membina dengan MYDS!

    ```tsx
    import { Button } from "@govtechmy/myds-react/button";

    export default () => {
      return <Button variant="primary-fill">Hello MYDS</Button>;
    };
    ```

</Step>
</Steps>
