## Usage

```tsx copy
import {
  ThemeProvider,
  useTheme,
  ThemeToggler,
} from "@govtechmy/myds-react/theme-toggler";
```

```tsx copy
export default () => (
  <ThemeProvider>
    <ThemeToggler />
  </ThemeProvider>
);
```

## Examples

### Basic Usage

Wrap your application with `ThemeProvider` and use `ThemeToggler` to switch between themes.
