## useTheme

Hook `useTheme` memberikan akses kepada tema semasa dan fungsi untuk menukar antara tema. Ia mesti digunakan dalam `ThemeProvider`.

## Pemasangan

Pastikan aplikasi anda dibungkus dengan `ThemeProvider` sebelum menggunakan `useTheme`. Anda boleh menggunakan komponen `<ThemeToggle />` yang disediakan atau `membina komponen tersuai` untuk menggunakan hook ini.

### Contoh: Susun Atur Utama

```tsx
import {
  ThemeProvider,
  ThemeToggler,
} from "@govtechmy/myds-react/theme-toggler";

export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="ms">
      <body>
        <ThemeProvider>
          <ThemeToggle />
          {children}
        </ThemeProvider>
      </body>
    </html>
  );
}
```

### Contoh: Susun Atur

```tsx
import {
  ThemeProvider,
  ThemeToggler,
} from "@govtechmy/myds-react/theme-toggler";

function Layout({ children }: { children: React.ReactNode }) {
  return (
    <ThemeProvider>
      <div className="app-container">
        <ThemeToggle />
        {children}
      </div>
    </ThemeProvider>
  );
}
```

## Menggunakan Hook useTheme

### Membina Komponen Tukar Tema Tersuai

Hook `useTheme` menyediakan cara mudah untuk mengurus status tema dalam aplikasi anda. Berikut adalah komponen tersuai yang membolehkan pengguna menukar antara mod cerah dan gelap.

```tsx
import { ThemeProvider, useTheme } from "@govtechmy/myds-react/hooks/use-theme";

const ThemeToggleCustom = () => {
  const { theme, toggleTheme } = useTheme();

  return (
    <button onClick={() => toggleTheme(theme === "light" ? "dark" : "light")}>
      Tukar ke mod {theme === "light" ? "gelap" : "cerah"}
    </button>
  );
};

export { ThemeToggleCustom, ThemeProvider };
```

### Mengintegrasikan Tukar Tema ke dalam Komponen Susun Atur

Untuk memastikan tema diterapkan di seluruh aplikasi, bungkus keseluruhan susun atur dengan `ThemeProvider`. Ini membolehkan semua komponen anak mengakses status tema dan fungsi penukaran tema.

```tsx
import { ThemeToggleCustom, ThemeProvider } from "./ThemeToggleCustom";

function Layout({ children }: { children: React.ReactNode }) {
  return (
    <ThemeProvider>
      <div className="app-container">
        <ThemeToggleCustom />
        {children}
      </div>
    </ThemeProvider>
  );
}
```

### Penjelasan

- Komponen `ThemeToggleCustom` mendapatkan tema semasa (`cerah` atau `gelap`) dan menyediakan butang untuk menukar antara kedua-duanya.
- Komponen `Layout` membungkus aplikasi dalam `ThemeProvider`, memastikan tema boleh diakses oleh semua komponen anak.
- Meletakkan `ThemeToggleCustom` dalam `Layout` membolehkan pengguna menukar tema dari mana-mana halaman.

Pendekatan ini menjadikan logik tema lebih modular, boleh digunakan semula, dan mudah disepadukan dalam pelbagai bahagian aplikasi.

**Nota:** `useTheme` mesti digunakan dalam `ThemeProvider`. Menggunakannya di luar akan menyebabkan ralat.

## API

`const { theme, toggleTheme } = useTheme();`

#### Parameter

Hook `useTheme` tidak menerima sebarang parameter.

#### Pulangan:

- **`theme: string`** – Tema semasa.
- **`toggleTheme: (theme: string) => void`** – Fungsi untuk menukar tema secara dinamik.
